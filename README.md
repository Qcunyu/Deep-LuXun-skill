##  项目概述

**Deep-LuXun-skill** 并非简单的语气模仿插件，而是一个深度的**认知框架复刻**。

本项目通过对 71,000 字的鲁迅原著、学术论文及对话录进行多轮“蒸馏”，将先生底层的**二难推理**、**归谬法**以及**国民性剖析逻辑**植入 LLM 的推理链中。它旨在提供一个冷峻、深刻且具有穿透力的视角，去剖析当代社会的文化现象与心理顽疾。

### 场景一：评“网络暴力与围观心智”
**用户提问：** 先生，如何看待现在的网络围观和言语暴力？
**Deep-LuXun 响应：**
> “凡是愚弱的国民，即使体格如何健全，如何茁壮，也只能做毫无意义的示众的材料和看客。……然而在网络这块沙地上，雪是不该下的。倘若以为躲在屏幕后的阴影里，便能通过投掷石块来获得道德的洁净，那大约是陷入了另一种阿Q式的胜利。这种狂欢，不过是无主名且无责任的‘示众’罢了。”

### 场景二：评“当代年轻人的躺平与挣扎”
**用户提问：** 大家都在说躺平，我也觉得努力没有意义。
**Deep-LuXun 响应：**
> “希望是附丽于存在的，有存在，便有希望，有希望，便是光明。但若是为了逃避那铁屋子里的闷热而索性闭目塞听，自以为是在‘躺平’，其实不过是在为那沉重的屋顶增加一两块瓦片。……我也并不劝人盲目地乐观，但我总觉得，与其在虚无中枯萎，不如做一点零碎的小事，即如萤火一般，也可以在黑暗里发一点光，不必等候炬火。”

##  核心特性

* **逻辑引擎**：内置 01-writings.md 调研得出的批判逻辑，善于捕捉论证中的悖论。
* **表达 DNA**：保留了先生特有的“大约、的确、然而、倘若”等辩证式句式，拒绝油滑。
* **多维语料支撑**：涵盖 1906-1936 年间的人生转折与思维进化记录。
* **绝对禁令**：严禁 AI 常见的“万金油”式正能量，保持冷峻的批判底色。

##  安装与使用

本项目适配 [Nuwa-Skill](https://github.com/alchaincyf/nuwa-skill) 框架及多种主流 Agent 工具。

1.  **下载仓库**：
    ```bash
    git clone https://github.com/Qcunyu/Deep-LuXun-skill.git
    ```
2.  **配置 Skill**：
    将 `SKILL.md` 文件内容复制并粘贴到你的 Agent 提示词配置区，或放入对应的 Skill 管理目录。
3.  **开始对话**：
    使用触发词（如：鲁迅、先生、批判性视角）启动深度逻辑。
